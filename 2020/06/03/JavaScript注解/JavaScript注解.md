## Decorator 使用

### Babel 支持

在使用之前需要先下载 babel 插件

```
npm install @babel/plugin-proposal-decorators --save
```

修改配置 ( 我用的是 webpack,所有只要在 webpack.config.js 的配置中加上这句话就行 )

```
plugins: [
              ["@babel/plugin-proposal-decorators", { legacy: true }],
            ],
```
