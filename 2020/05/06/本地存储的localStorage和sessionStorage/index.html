иҝҷзҜҮж–Үз« жңҖеҗҺдҝ®ж”№дәҺ еӨ©д№ӢеүҚгҖӮ жң¬йЎөеҶ…е®№еҸҜиғҪе·Із»ҸиҝҮж—¶пјҒ

localStorage(жң¬ең°еӯҳеӮЁ)

localStorage жҳҜеҸӘиҜ»зҡ„гҖӮж•°жҚ®еӯҳеӮЁд№ҹжҳҜи·ЁжөҸи§ҲеҷЁдјҡиҜқгҖӮж•°жҚ®еӮЁеӯҳеңЁ localStorage жҳҜж— жңҹйҷҗзҡ„гҖӮ
еёёз”Ёзҡ„еӣӣдёӘ API д№ҹеҫҲз®ҖеҚ• пјҡ

дҪҶйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢеңЁз§»еҠЁи®ҫеӨҮдёҠзҡ„жөҸи§ҲеҷЁжҲ–еҗ„ Native App з”ЁеҲ°зҡ„ WebView йҮҢпјҢlocalStorage йғҪжҳҜдёҚеҸҜйқ зҡ„пјҢеҸҜиғҪдјҡеӣ дёәеҗ„з§ҚеҺҹеӣ пјҲжҜ”еҰӮиҜҙйҖҖеҮә AppгҖҒзҪ‘з»ңеҲҮжҚўгҖҒеҶ…еӯҳдёҚи¶ізӯүеҺҹеӣ пјүиў«жё…з©әгҖӮ

sessionStorage(дјҡиҜқеӯҳеӮЁ)

еҪ“йЎөйқўиў«е…ій—ӯж—¶,ж•°жҚ®еӯҳеӮЁеңЁ sessionStorage дјҡиў«жё…йҷӨ

localStorage дёҖдәӣдёҚдёәдәәзҹҘзҡ„ж–№жі•

localStorage.hasOwnProperty() жЈҖжҹҘ localStorage дёӯеӯҳеӮЁзҡ„ж•°жҚ®йҮҢжҳҜеҗҰдҝқеӯҳжҹҗдёӘеҖј

storage дәӢд»¶

еҪ“еӯҳеӮЁж•°жҚ®еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢдјҡи§ҰеҸ‘ storage дәӢд»¶гҖӮеҖјеҫ—зү№еҲ«жіЁж„Ҹзҡ„жҳҜпјҢиҜҘдәӢд»¶дёҚеңЁеҜјиҮҙж•°жҚ®еҸҳеҢ–зҡ„еҪ“еүҚйЎөйқўи§ҰеҸ‘гҖӮеҰӮжһңжөҸи§ҲеҷЁеҗҢж—¶жү“ејҖдёҖдёӘеҹҹеҗҚдёӢйқўзҡ„еӨҡдёӘйЎөйқўпјҢеҪ“е…¶дёӯзҡ„дёҖдёӘйЎөйқўж”№еҸҳ sessionStorage жҲ– localStorage зҡ„ж•°жҚ®ж—¶пјҢе…¶д»–жүҖжңүйЎөйқўзҡ„ storage дәӢд»¶дјҡиў«и§ҰеҸ‘пјҢиҖҢеҺҹе§ӢйЎөйқўе№¶дёҚи§ҰеҸ‘ storage дәӢд»¶гҖӮеҸҜд»ҘйҖҡиҝҮиҝҷз§ҚжңәеҲ¶пјҢе®һзҺ°еӨҡдёӘзӘ—еҸЈд№Ӣй—ҙзҡ„йҖҡдҝЎгҖӮпјҲеҪ“з„¶ ie иҝҷдёӘзү№дҫӢйҷӨеӨ–пјҢе®ғеҢ…еҗ«иҮӘе·ұжң¬дәӢд№ҹдјҡи§ҰеҸ‘ storage дәӢд»¶пјүгҖӮ

event.key еұһжҖ§пјҡеұһжҖ§еҖјдёәеңЁ session жҲ– localStorage дёӯиў«дҝ®ж”№зҡ„ж•°жҚ®й”®еҖјгҖӮ
event.oldValue еұһжҖ§пјҡеұһжҖ§еҖјдёәеңЁ sessionStorage жҲ– localStorage дёӯиў«дҝ®ж”№зҡ„еҖјгҖӮ
event.newValue еұһжҖ§пјҡеұһжҖ§еҖјдёәеңЁ sessionStorage жҲ– localStorage дёӯиў«дҝ®ж”№еҗҺзҡ„еҖј
event.url еұһжҖ§пјҡеұһжҖ§еҖјдёәдҝ®ж”№ sessionStorage жҲ– localStorage дёӯеҖјзҡ„йЎөйқўзҡ„ URL ең°еқҖ
event.storageArea еұһжҖ§ : еұһжҖ§еҖјдёәиў«еҸҳеҠЁзҡ„ sessionStorage еҜ№иұЎжҲ– localStorage еҜ№иұЎ

еҺҹзҗҶ

еңЁ HTML5 дёӯпјҢжң¬ең°еӮЁеӯҳжҳҜдёҖдёӘ window зҡ„еұһжҖ§пјҢеҢ…жӢ¬ localStrong е’Ң sessionStrongгҖӮlocalStrong е’Ң sessionStrong йғҪжҳҜеӮЁеӯҳеңЁ window йҮҢзҡ„ strong е®һдҫӢеҜ№иұЎпјҢжҲ‘д»¬еҜ№ Window.sessionStorage е’Ң Window.localStorage еұһжҖ§дҪҝз”Ёе…¶е®һе°ұжҳҜж”№еҸҳдәҶ window жң¬ең°еӯҳеӮЁзҡ„еҜ№иұЎгҖӮ
Window еҜ№иұЎиЎЁзӨәдёҖдёӘжөҸи§ҲеҷЁзӘ—еҸЈжҲ–дёҖдёӘжЎҶжһ¶гҖӮеңЁе®ўжҲ·з«Ҝ JavaScript дёӯпјҢWindow еҜ№иұЎжҳҜе…ЁеұҖеҜ№иұЎпјҢжүҖжңүзҡ„иЎЁиҫҫејҸйғҪеңЁеҪ“еүҚзҡ„зҺҜеўғдёӯи®Ўз®—гҖӮ
